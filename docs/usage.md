# 安装包目录结构

```yaml
deployer:
	packages:
		- fuel mirrors
		- docker packages
		- pypi packages
		- files packages
		- application(docker)
	sources:
		- fuel-additive
		- fuel-mirrors
		- kubespray
		- steam
		- boot
	install.sh

```

install fuel master
install additive
create cluster
add node
config node
provision
deployment

install additvie
- push packages
- install additive
